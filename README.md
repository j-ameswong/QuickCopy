# Quick Image Copy Browser Extension

A simple browser extension to allow for `CTRL + C` to work on hovered images.

The only option for the extension is the activation key, the mod key is hard-coded to `CTRL`

## Compatability

Tested on Brave and Chrome, works on Firefox but extension is not persistent across browser sessions.

## Installation

1. Go to the releases page and download `quickcopy-v1.0.zip`
2. Unzip the file to a folder on your computer
3. Open Chrome and go to `chrome://extensions`
4. Turn on Developer Mode (toggle button on top right)
5. Click Load Unpacked and select the unzipped folder
